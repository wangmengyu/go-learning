1. 协程是 轻量级 的线程
2. 非抢占式多任务处理，由协程主动交出控制权
3. 编译器，解释器，虚拟机层面的多任务
4. 多个协程可以在一个或者多个线程上运行


查看数据冲突：
go run -race xxx.go

子程序是个协程的一个特例

普通的子函数
子函数执行完了把控制权交给调用它的main函数

协程：
main函数和，work函数有双向通道，可以双向的交换控制权，和数据。相当于并发执行的两个线程

任何函数只要加上go就能给调度器运行，就是个协程
